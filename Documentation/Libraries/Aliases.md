# Aliases
Each library contains shortened versions of the same functions. These are called aliases. Not every function has an alias, but the ones that do are listed below.

## Globals
`tolist` -> `tL`

## List

`map` -> `m`\
`fold` -> `f`\
`reduce` -> `r`\
`cumulate` -> `c`\
`reverse` -> `R`\

`clone` -> `C`\
`sort` -> `S`\
`sum` -> `s`\
